# Week 06 — Vision Transformer (ViT)

## What was covered

- Turning images into patch tokens
- Introducing the CLS token for global image representation
- Applying transformer encoder blocks to visual tokens

## Key insights

- Patch embedding is analogous to token embedding in NLP
- CLS token acts like an aggregation token for classification
- ViT reuses standard transformer blocks with image-specific input preparation

## What was confusing

- Why ViT often needs larger datasets or strong augmentation
- How patch size affects compute cost and representation granularity
